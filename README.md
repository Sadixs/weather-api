# Weather App

A sleek and modern weather application that provides real-time weather data for any city worldwide. Powered by the OpenWeather API.

## Features

- **Search by City** - Get instant weather updates for any city globally
- **Real-time Data** - Live temperature and weather conditions
- **Smooth Animations** - Elegant fade-in transitions for weather results
- **Responsive Design** - Clean gradient interface that works on all devices

## Prerequisites

- A modern web browser
- OpenWeather API key (free tier available)

## Installation

1. Clone the repository or download the files
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Replace `OPENWEATHER_FREEAPI_HERE` in `js/script.js` with your actual API key
4. Open `index.html` in your browser

## Usage

1. Enter a city name in the search input field
2. Click the "Search" button
3. View the weather information including:
   - City name
   - Current temperature (°C)
   - Weather description

## Technologies Used

| Technology | Purpose |
|------------|---------|
| **HTML5** | Structure and semantics |
| **CSS3** | Styling, gradients, and fade-in animations |
| **JavaScript (ES6)** | API integration, async/await, and DOM manipulation |
| **OpenWeather API** | Weather data source |

## API Reference

This app uses the OpenWeather Current Weather Data API:
- Endpoint: `https://api.openweathermap.org/data/2.5/weather`
- Units: Metric (Celsius)
